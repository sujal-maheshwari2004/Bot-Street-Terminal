# Bot Street Terminal

Bloomberg-inspired trading terminal for the Bot Street distributed market simulator.

Built with React, Vite, and Recharts.

Deployed on Google Cloud Run.

Live Demo: [Bot Street Terminal](https://bot-street-terminal-957519594640.us-central1.run.app/?utm_source=chatgpt.com)

---

# Overview

Bot Street Terminal is a real-time trading workspace designed to visualize a Kafka-native algorithmic market simulation.

The frontend consumes data from the FastAPI backend and renders:

* live candlestick charts
* order books
* trade feeds
* sentiment signals
* portfolio leaderboards
* market indicators
* trading interfaces

The UI architecture is inspired by professional trading systems and terminal workflows.

# UI Philosophy

The terminal intentionally avoids consumer-dashboard styling.

The interface is designed around:

* dense information display
* low-latency readability
* professional terminal ergonomics
* high signal-to-noise ratio
* keyboard-and-chart centric workflows

Visual inspiration comes from:

* Bloomberg Terminal
* Eikon
* proprietary trading dashboards
* crypto execution terminals

---

# Architecture

```text
React Terminal
      │
      ▼
FastAPI Gateway
      │
      ▼
Kafka Event Infrastructure
```

The frontend itself remains stateless.

All market state originates from Kafka-backed backend services.
